FabricEye - Fabric Defect Detection Using CNN
Description

Project developed as part of my Mechatronics Engineering training.

This system uses Convolutional Neural Networks (CNN) to automatically detect fabric defects through computer vision and image processing techniques.

The project applies Artificial Intelligence concepts to improve quality inspection processes in the textile industry.

Objective

To automate the textile quality inspection process by detecting defects in fabric images, reducing human inspection errors and improving manufacturing efficiency.

Technologies Used
Python
TensorFlow
Keras
OpenCV
NumPy
Matplotlib
Applications
Textile quality control
Industry 4.0
Smart Manufacturing
Computer Vision
Artificial Intelligence applied to industrial processes
Results

The CNN model was trained using fabric images with and without defects to perform automatic classification.

The project demonstrates the potential of Deep Learning techniques for industrial inspection and manufacturing optimization.
